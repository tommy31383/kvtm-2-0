UI/UX System · Cozy Match-3 / Diary Game (v2.5 PIVOT)

⚠️ v2.5 PIVOT: Royal Match navy/gold + Lilita One direction (v2.4) ABANDONED. New canonical: Cozy Match-3 / Diary Game Design System. Source of truth: kvtm_2_0_design_system.md
📜 6 DESIGN PRINCIPLES (canonical)
  1. 1. Cozy, not loud — Desaturated warm palette. No pure white, no pure black, no high-saturation primaries.
  2. 2. Handwritten, not typeset — Display + body fonts handwritten. Slight rotation -1° to +1° on every interactive element. Personal journal feel.
  3. 3. Chunky silhouettes — Every interactive surface: thick warm-brown stroke + hard drop shadow + pillow highlight. NO flat web buttons.
  4. 4. One color, one job — Each accent owns exactly one role. Don't mix roles.
  5. 5. Scene is the hero — UI hugs edges. Illustration owns the middle.
  6. 6. Diary mood — Narration is part of loop. Reward = a line in the diary, not just a coin number.

📔 DIARY LOOP (cozy game core)
  1. 1. Morning page — Caveat Brush chapter title + Gloria Hallelujah narration on banner card. Sets today's task.
  2. 2. Play — Sort/Triple board. Pieces higher saturation than UI but stay in cozy hue family.
  3. 3. Reward = a line in diary — NOT just "+50 coins". Win screen reveals one new line of narration.
  4. 4. Decoration unlock — placing object triggers sticker animation in diary.
  5. 5. Mood marker — end of session player picks 1 of 5 stickers (sun/rain/heart/flower/star) = "rate this session".
🗣️ TONE OF VOICE
✅ DO
"Day 12. The fountain finally works. Mira left a note on the porch."
"Tap to plant the lavender →"
Conversational · present-tense · gentle. Player narrates as themselves. Numbers exist but never the headline — line of story is.
❌ DON'T
"DAILY DEAL — 50% OFF — ENDS SOON"
"VICTORY! +500 XP"
Friendly, never urgent. NO LIMITED TIME · BUY NOW · all-caps urgency.
❌ DON'T LIST (enforce strictly)
  • ❌ Pure white #fff or pure black #000 anywhere
  • ❌ System sans-serif (Inter / Roboto / Helvetica) for player-facing copy
  • ❌ Saturated brand-gamey blue/red — keep palette warm + muted
  • ❌ Perfectly straight elements — always nudge ±0.3 to ±1.2 degrees
  • ❌ Multiple accent colors competing for the same role
  • ❌ Blurred shadows — only hard offset shadows
  • ❌ Web-style flat buttons or material elevation
  • ❌ Filler content / data slop — every element earns its place
