## 🏗️ Architecture overview

### Bloom data format evolution (3 phases)

```
PHASE 0 — Legacy:
  _BLOOM_RECTS[c] = [[x,y,w,h,dx,dy], ...]  // 10 frames, single anim per color
  _BLOOM_DURS[c]  = null | number | [d0..d9]

PHASE 1 — Multi-variant (nested):
  _BLOOM_RECTS[c] = [ [[r0..r9]], [[r0..r9]], ... ]  // variants[role_idx]
  _BLOOM_DURS[c]  = [v0_dur, v1_dur, ...]
  Role: 0=Bloom, 1=Nụ, 2=Bông, 3+=reserved

PHASE 2/3 — Atlas + Module + Anim (current):
  _BLOOM_DATA[c] = {
    modules: {
      <id>: { x, y, w, h, [dx, dy, px, py] }
    },
    anims: {
      bloom:  [{m:<id>, d:<ms>}, ...],
      bud:    [{m:<id>, d:<ms>}, ...],
      flower: [{m:<id>, d:<ms>}, ...]
    }
  }
  Engine auto-fallback: _BLOOM_DATA → _BLOOM_RECTS → static img
```

**Engine resolver** at [engine/sort_blossom_render.js:_probeBloom](engine/sort_blossom_render.js): prefers _BLOOM_DATA → falls back legacy → produces unified `variants[]` shape → consumers zero-touch.

### Render path roles
- `_bloomSheet(imgEl, color)` — bloom transition (variants[0]=Bloom). Uses rAF + cumulative time table.
- `upgradeQueueBuds(cell)` — anim Nụ (variants[1]). Loop if ≥2 frames.
- `upgradeActiveFlowers(cell)` — anim Bông (variants[2]). Loop if ≥2 frames.
- `_startIdleLoop(cv, variant, drawFn)` — shared rAF scheduler powering Nụ + Bông idle.

### Pivot system (Phase 3)
- Module schema adds optional `px, py` (anchor in module-local coords)
- Default `(w/2, h)` bottom-center = matches legacy
- `_getRectPivot(rect)` returns effective pivot
- `_bloomSheet drawFrame`: `ox = refX - pivotDrawX, oy = refY - pivotDrawY`
- Static paths (queue + active): canvas h = max(refAboveExt, frameAboveExt), pivot at canvas bottom

## 🛠️ Tool UI cheatsheet

### Tabs
- **🎬 Anim** — compose anim bằng pick module; timeline frames
- **📦 Modules** — define module library (sheet + list panel)
- **🪴 Layout** — pot preview + CSS var sliders

### Anim tab — Sheet interactions
- Click empty → deselect
- Click module khác → swap selected frame's modId
- Click selected frame's module body → drag MOVE module (propagates to all frames ref)
- Click selected frame's module corner → drag RESIZE module
- Shift+click bất kỳ module → +frame mới ref module đó
- Bottom strip: thumbnail library — click = add frame, shift+click = always add

### Modules tab — Sheet interactions
- Drag empty → CREATE new module (`_manual: true`, không bị auto-clean)
- Drag corner of selected → RESIZE module
- Drag body of selected → MOVE module
- **Shift+click trong rect** → SET PIVOT tại điểm click
- Delete key → xoá module (warn if refs > 0)
- Auto-dedup button → gộp identical rects
- 🧹 Clean orphans → drop module 0-ref

### Pivot
- Default (no explicit) = `(w/2, h)` bottom-center → dim cyan ⊕
- Explicit (set qua shift+click hoặc input) → orange ⊕
- Edit pane inputs: pX, pY (module-local coords)
- Buttons: `↺ Default` (clear) / `⊥ Center-bot` (preset)
- Tool center preview: frame CENTERED + pivot ⊕ overlaid at pixel
- Pivot saved trong `_BLOOM_DATA[color].modules[id].px/py`

### Debug hooks (DevTools console)
```js
_sbClearBloomCache()       // wipe cache (after data change)
_sbClearBloomCache('R')    // wipe 1 color
_setIdleSpeed(2)           // 2× slower idle anims
_setIdleSpeed(0.5)         // 2× faster
_pauseIdleAnims()          // freeze all idle
_resumeIdleAnims()
_idleAnimCount()           // → number of active instances
```
