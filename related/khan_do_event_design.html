Khác biệt cốt lõi của KHĂN ĐỎ vs Garden Design Makeover
Garden Design Makeover (gốc)
  • • Challenge = Match-3 puzzle độc lập
  • • Reward sao = swap kẹo / kết hợp tile
  • • Đảo riêng, không tích hợp core game
  • • Story generic (Melissa và một garden)
  • • Theme decor đa quốc gia (Pháp / Ý / Mỹ)
KHĂN ĐỎ (KVTM event)
  • ✓ Challenge = nộp nông sản KVTM (cây / vật / sản phẩm chế biến)
  • ✓ Reward sao = từ chính core farm loop của user
  • ✓ Tích hợp sâu vào KVTM economy → user phải chăm sóc vườn chính để có nông sản nộp event
  • ✓ Story drama Việt Nam đặc trưng (Khăn Đỏ + Phú + tiểu thư)
  • ✓ Theme decor 100% Việt (Mansion thành phố / Phố cổ Hà Nội / Biệt phủ làng quê / Cung đình Huế)
lightbulb
Insight chiến lược

Bằng cách dùng nông sản + đơn hàng + sản phẩm chế biến KVTM làm currency của event (thay vì match-3), event này biến thành amplifier cho core farm loop chứ không phải mini đảo riêng. User phải trồng cây · giao đơn · sản xuất nhiều hơn → tăng session và usage tài nguyên KVTM → tăng KC sink (skip timer trồng cây · mua hạt giống premium · skip máy chế biến...). Đây là compounding effect mà Gardenscapes không có vì match-3 đứng độc lập.

Visual Reference · Before/After Pattern
BEFORE — Đổ nát
Sân khấu nhạc đổ nát · gạch lộn xộn · bụi bặm · vỏ chai vứt · Khăn Đỏ với cây chổi
PATTERN HOOK

AFTER — Lộng lẫy
Sân khấu nhạc rực vàng · cây xanh · sàn checkered · đèn dây · DJ deck · trang trí lễ hội
📌 Đây là core visual hook em apply cho 4 maps Khăn Đỏ (Mansion thành phố · Phố cổ Hà Nội · Biệt phủ làng quê · Cung đình Huế) — mỗi map có 3 state transformation từ ruin → mid → glorious. User chứng kiến vườn "sống lại" qua bàn tay Khăn Đỏ → emotional ownership + share viral before-after Facebook.

Challenge type · 3 hình thức
A · Nộp nông sản
Yêu cầu nộp số lượng cây / vật / sản phẩm chế biến từ kho KVTM
Ví dụ: "Nộp 100 cà chua + giao 30 đơn hàng → 50 sao + 1 decor random"
Tỉ lệ: 60% challenge
B · Quest combo
Hoàn thành nhiều hành động trong khoảng thời gian (trồng X cây + giao Y đơn hàng + sản xuất Z sản phẩm chế biến)
Ví dụ: "Trồng 20 hoa mai + giao 30 đơn hàng + sản xuất 10 bánh trong 24h"
Tỉ lệ: 30% challenge
C · Mini puzzle
Mini game ngắn 30-60s ngay trong vườn (drag, sort, match) để kiếm bonus sao
Ví dụ: "Drag 5 hoa đúng vị trí trong 30s → 20 sao bonus"
Tỉ lệ: 10% challenge
